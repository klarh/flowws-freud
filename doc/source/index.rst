`flowws-freud <https://github.com/klarh/flowws-freud>`_ is an
in-development library to create workflows for analysis of simulations
using `freud <https://freud.readthedocs.io>`_.

`flowws-freud` only contains stages for workflows; to see how to
use the stages, consult the `flowws documentation
<https://flowws.readthedocs.io>`_.

Installation
============

Install `flowws-freud` from source on github::

  pip install git+https://github.com/klarh/flowws-freud.git#egg=flowws-freud
